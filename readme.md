# 这是一个作业工程

以后所有的作业都是在这个工程下提交。

提交规则
 * 每次作业需要独立开辟单独的分支，禁止在原来的分支上就修改和二次提交
 * 可以参考，但是禁止复制粘贴其他人代码 禁止使用网站源码进行提交
 * 你可以使用ai生成，但是如果提问自己不明白，一律认为没做作业处理

作业分之创建规则：
 * 分支创建必须还有自己的姓名拼音，不允许拼音缩写或者英文或者日语名称
 * 分支起名格式：比如`6月4日`发布的作业，那么分支创建名称就是 `feature-xxxx-6-4`
 * 所有每次作业的分支必须以main为主分支开辟自己的作业分支。
